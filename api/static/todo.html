Expected input is JSON with a "points" list of one or more tasks. Valid input for this protocol will create a project (if needed) containing all tasks in the "tasks" dictionary: 
{
  "note": "Project note",
  "group": "Project group",
  "source": "Neu3",
  "points": [{"Pos": [14049,14237,17567],
              "Prop": {"action": "to split",
                       "user":"shinomiyaa"},
              "priority": "2"},
             {"Pos": [24049,24237,17567],
              "Prop": {"action": "to split",
                       "user":"shinomiyaa"},
              "priority": "2"},
             {"Pos": [34049,34237,17567],
              "Prop": {"action": "to split",
                       "user":"neacee"}
             }
           ]
}
Elements in gray are optional, all others are required. Any additional data not shown above is ignored.
